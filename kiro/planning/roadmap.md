# Project Roadmap

## Phase 1: Research & Planning
- Studied real-world public security challenges
- Identified threat categories and system requirements
- Selected suitable AI models and technologies

## Phase 2: Architecture Design
- Designed end-to-end system flow
- Defined backend, frontend, AI, and database roles
- Planned alerting and analytics mechanisms

## Phase 3: Backend Development
- Built APIs using FastAPI
- Integrated live camera feed using Python and OpenCV
- Implemented YOLO for object detection
- Used SORT for object tracking
- Stored detections in MongoDB

## Phase 4: Frontend Development
- Built dashboard using React
- Displayed live detections and analytics
- Implemented export and reporting features

## Phase 5: Testing & Optimization
- Reduced false positives
- Improved inference speed
- Validated alert logic
